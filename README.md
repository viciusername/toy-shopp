# ToyPlanet — Магазин іграшок 

## Опис
ToyPlanet – навчальний проект для лабораторних робіт з курсу C#. Включає консольну програму, бізнес-логіку, юніт-тести, а також приклади деплою, веб-інтерфейсу, роботи з різними СУБД та REST API.

## Архітектура
- **ToyPlanet.Core** — бізнес-логіка, моделі, сервіси
- **ToyPlanet.Tests** - юніт-тести (xUnit)
- **ToyPlanet.App** — консольний додаток (демонстрація роботи)
- **Vagrantfile** - автоматизація деплою
