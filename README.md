# ToyPlanet — Магазин игрушек

## Описание
ToyPlanet — учебный проект для лабораторных работ по курсу C#. Включает консольное приложение, бизнес-логику, юнит-тесты, а также примеры деплоя, веб-интерфейса, работы с разными СУБД и REST API.

## Архитектура
- **ToyPlanet.Core** — бизнес-логика, модели, сервисы
- **ToyPlanet.Tests** — юнит-тесты (xUnit)
- **ToyPlanet.App** — консольное приложение (демонстрация работы)
- **Vagrantfile** — автоматизация деплоя

## Быстрый старт (Lab 1)
1. Склонируйте репозиторий
2. Соберите и запустите консольное приложение:
   ```
   dotnet run --project src/ToyPlanet.App/ToyPlanet.App.csproj
   ```
3. Запустите тесты:
   ```
   dotnet test
   ```

## Структура
- `src/` — исходный код
- `tests/` — тесты
- `vagrant/` — файлы для деплоя

## Возможности
- Категории, товары, корзина, заказы
- Сервис расчёта скидок
- Тесты для всех основных сценариев

## Дальнейшие этапы
- NuGet-пакет, деплой через Vagrant
- ASP.NET Core MVC + OAuth2
- EF Core, поддержка 4 СУБД
- REST API, Swagger, интеграционные тесты
