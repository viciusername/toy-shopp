# Деплой через Vagrant (Lab 2)

## 1. Сборка NuGet-пакета

Выполните в папке src:
```
dotnet pack ToyPlanet.Core.csproj -o ../nupkg
```

## 2. Использование Vagrantfile

- Vagrantfile разворачивает приложение на Ubuntu и Windows.
- Внутри VM клонируется репозиторий, устанавливается .NET SDK, проект билдится и запускается.
- Для приватного NuGet-репозитория используйте BaGet или скачайте nupkg вручную.

## 3. Запуск

1. Перейдите в папку vagrant:
```
cd vagrant
vagrant up
```
2. После завершения провижининга приложение будет запущено в обеих VM.
